# Nguemoue DB Objects Migrator

**Nguemoue DB Objects Migrator** est un package Laravel 12.x permettant de gérer la migration versionnée d'objets de base de données (**fonctions**, **procédures stockées**, **vues**, **triggers**, etc.) pour MySQL/MariaDB. Il complète les migrations Laravel classiques en offrant un moyen d'inclure ces objets dans le contrôle de version et le déploiement.

## Installation

Installez le package via Composer :

```bash
composer require nguemoue/db-objects-migrator
